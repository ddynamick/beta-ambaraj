
# Deployment Guide

## Frontend
1. Open terminal in frontend folder
2. Run:
   npm install
   npm run build
   npm run start

## Backend
1. Open terminal in backend folder
2. Run:
   npm install
   npm start

## GitHub Upload
1. Create repository
2. Upload frontend and backend folders
3. Commit changes

## GoDaddy VPS
- Install Node.js
- Install PM2
- Configure Nginx
- Point domain to VPS
